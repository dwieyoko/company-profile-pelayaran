Sertifikasi & Standar

Seluruh armada kami memiliki sertifikasi internasional dan mematuhi standar keselamatan maritim

ISM Code

International Safety Management Code untuk sistem manajemen keselamatan

ISPS Code

International Ship and Port Facility Security Code untuk keamanan kapal

MLC 2006

Maritime Labour Convention untuk standar kerja maritim

MARPOL

International Convention for Prevention of Pollution from Ships

Perawatan & Maintenance

Kami menerapkan program perawatan preventif yang ketat untuk memastikan seluruh armada dalam kondisi prima dan siap operasi.
